Сұрыптау (Sorting)

Бұл сабақта Python тілінде тізім элементтерін өсу және кему ретімен сұрыптауды, list.sort() әдісін, sorted() функциясын және сұрыптауға арналған практикалық есептерді қарастырамыз.

Сабақ соңында сен сыныптағы оқушылардың бағаларын, сандар тізімін немесе кез-келген деректерді өсу/кему ретімен реттей алатын боласың.

Оқу мақсаты

  • Сұрыптау ұғымын түсіну және программалаудағы маңызын түсіндіру;
  • list.sort() әдісін өсу және кему ретімен қолдану;
  • sorted() функциясының ерекшелігін білу;
  • Бірөлшемді массив/тізім элементтерін өсу және кему ретімен шығару;
  • Сұрыпталған тізімнен K-ші үлкен элементті табу есебін шығару.

1-бөлім · Теория

Сұрыптау деген не?

Программалауда сұрыптау — бұл массив немесе тізім элементтерін белгілі бір ретпен (көбіне өсу немесе кему ретімен) орналастыру процесі. Мысалы, журналдағы бағаларды кішіден үлкенге, не аты-жөндерді алфавиттік ретпен реттеу.

Python-да сандар, жолдар (string), тіпті күрделірек объектілер де сұрыптала алады, егер оларды салыстыру ережесі белгілі болса.

1) list.sort() әдісі

list.sort() — тізімді орнында (in-place) өзгертеді, яғни бастапқы тізімнің өзі сұрыпталады. 

numbers = [3, 2, 7]

numbers.sort()           # өсу ретімен
print(numbers)           # [2, 3, 7]

numbers.sort(reverse=True)  # кему ретімен
print(numbers)              # [7, 3, 2]
  • reverse=False (әдепкі) → өсу ретімен;
  • reverse=True → кему ретімен.

2) sorted() функциясы

sorted(iterable) — берілген тізімнің немесе басқа қайталанатын объектінің жаңа сұрыпталған көшірмесін қайтарады. Бастапқы тізім өзгермейді. 

arr = [3, 2, 7]

new_arr = sorted(arr)        # өсу ретімен жаңа тізім
print("new_arr:", new_arr)   # [2, 3, 7]
print("arr:", arr)           # [3, 2, 7] бастапқысы өзгермейді

desc = sorted(arr, reverse=True)
print("desc:", desc)         # [7, 3, 2]

3) Сандар мен жолдарды сұрыптау

Сандарды өсу ретімен сұрыптау:

scores = [75, 90, 60, 100, 85]
scores.sort()
print(scores)  # [60, 75, 85, 90, 100]

Жолдарды (мәтіндерді) алфавиттік ретпен сұрыптау:

names = ["Айбек", "Бота", "Дана", "Арман"]
names.sort()
print(names)   # ['Айбек', 'Арман', 'Бота', 'Дана']

Күрделі алгоритмдер (көпіршік сұрыптау, кірістіру арқылы сұрыптау, таңдау арқылы сұрыптау) да бар, бірақ Python-ның sort() және sorted() функциялары оларды біздің орнымызға орындайды.

2-бөлім · Мысалдар

1-мысал · A[N] массивін өсу және кему ретімен шығару

A[N] бірөлшемді массиві берілген. Осы массивтің элементтерін алдымен өсу ретімен, кейін кему ретімен экранға шығар. 

A = [8, 3, 5, 1, 9, 4]

print("Бастапқы тізім:", A)

# Өсу ретімен
A_sorted_asc = sorted(A)
print("Өсу ретімен:", A_sorted_asc)

# Кему ретімен
A_sorted_desc = sorted(A, reverse=True)
print("Кему ретімен:", A_sorted_desc)

Программа нәтижесінің үлгісі:

Бастапқы тізім: [8, 3, 5, 1, 9, 4]
Өсу ретімен: [1, 3, 4, 5, 8, 9]
Кему ретімен: [9, 8, 5, 4, 3, 1]

2-мысал · K-ші үлкен элементті табу

B[N] бірөлшемді, бірдей екі элементі жоқ массив берілген (0 < N < 100). Осы массивтегі K-ші үлкен элементті тап.

Идея:

  1. Тізімді кему ретімен сұрыптаймыз;
  2. К-ші үлкен элемент индексі — K - 1.
B = [8, 14, 52, 25, 35, 67, 41, 30]
K = 2  # 2-ші үлкен элемент

# 1) Кему ретімен сұрыптау
B_sorted = sorted(B, reverse=True)

print("Кему ретімен:", B_sorted)
print(f"{K}-ші үлкен элемент:", B_sorted[K - 1])

Программа нәтижесінің үлгісі:

Кему ретімен: [67, 52, 41, 35, 30, 25, 14, 8]
2-ші үлкен элемент: 52

Егер K мәнін пайдаланушы енгізсе, онда input() арқылы оқып, бүтін санға (int) түрлендіріп аламыз. 

K = int(input("Қайыншы элементті тапқың келеді (K): "))

B_sorted = sorted(B, reverse=True)
print(B_sorted[K - 1])

3-бөлім · Практика

Тапсырма 1 · Бағаларды өсу ретімен

Пайдаланушыдан бірнеше оқушының бағаларын енгізетін, кейін оларды өсу ретімен экранға шығаратын бағдарлама жаз.

Үлгі кіріс/шығыс:

Кіріс (input) Шығыс (output)
5 3 4 2 5 2 3 4 5 5

Тапсырма 2 · Ең кіші және ең үлкен элемент

Берілген тізімдегі элементтерді өсу ретімен сұрыптап, оның ең кіші және ең үлкен элементін экранға шығар.

Үлгі кіріс/шығыс:

Кіріс Шығыс
[10, 3, 25, 7] Ең кіші: 3, Ең үлкен: 25

Тапсырма 3 · Аты-жөндерді алфавит бойынша реттеу

Пайдаланушыдан 5 оқушының атын енгізетін және оларды алфавиттік ретпен сұрыптап шығаратын бағдарлама құр.

5-бөлім · Қорытынды

  • Сұрыптау — деректерді түсінікті, реттелген күйде көрсету үшін өте маңызды;
  • list.sort() тізімді орнында өзгертеді;
  • sorted() — жаңа сұрыпталған көшірме қайтарады;
  • Кему ретімен алу үшін reverse=True параметрі қолданылады;
  • Сұрыптау идеясы олимпиада есептерінде, жобаларда, өмірдегі көптеген тапсырмаларда қолданылады.
