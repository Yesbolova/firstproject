Таңдау операторлары

Бұл сабақта Python тілінде таңдау операторларын ( if, elif, else ) қолдануды, бірнеше жағдайдың ішінен біреуін таңдауды және мәзір түріндегі программалар жазуды үйренеміз.

Сабақ соңында сен күрделі таңдау логикасын түсініп, 10 сұрақтық Mini-Quiz арқылы біліміңді тексере аласың.

Оқу мақсаты

  • Таңдау операторының (if, elif, else) мәнін түсіну;
  • Бірнеше тармақтан тұратын таңдау құрылымын жазу;
  • Сан, баға, күн, мәзір сияқты жағдайларда таңдау операторларын қолдану;
  • Қарапайым «таңдау менюі» бар программалар жазу;
  • Mini-quiz арқылы тақырыпты бекіту.

1-бөлім · Теория

Таңдау операторы деген не?

Көп жағдайда программа бірнеше нұсқаның бірін таңдап орындауы керек болады. Мысалы: баға қою, аптаның күнін анықтау, менюдегі пунктті таңдау. Осындай кезде таңдау операторлары қолданылады.

Python-та негізгі таңдау операторы — if, оған қосымша elif және else қолданылады: 

if шарт1:
    # егер шарт1 ақиқат болса
    ...
elif шарт2:
    # әйтпесе, егер шарт2 ақиқат болса
    ...
else:
    # алдыңғы барлық шарттар жалған болса
    ...

if–elif–else мысалы

Баға қою логикасы:

score = int(input("Ұпайды енгіз: "))

if score >= 90:
    grade = "A"
elif score >= 75:
    grade = "B"
elif score >= 60:
    grade = "C"
elif score >= 50:
    grade = "D"
else:
    grade = "F"

print("Сенің бағаң:", grade)
Кіріс: 83
Шығыс: Сенің бағаң: B

Бірнеше жағдайды таңдау

Мысалы, аптаның реттік номеріне қарай оның атауын шығару:

day = int(input("Күн номерін енгіз (1-7): "))

if day == 1:
    print("Дүйсенбі")
elif day == 2:
    print("Сейсенбі")
elif day == 3:
    print("Сәрсенбі")
elif day == 4:
    print("Бейсенбі")
elif day == 5:
    print("Жұма")
elif day == 6:
    print("Сенбі")
elif day == 7:
    print("Жексенбі")
else:
    print("Мұндай күн номері жоқ")

Меню (таңдау пункттері)

Пайдаланушыға бірнеше нұсқа ұсынатын меню:

print("1) Қосу")
print("2) Алу")
print("3) Шығу")

choice = input("Таңдауыңды енгіз (1/2/3): ")

if choice == "1":
    print("Қосу операциясы")
elif choice == "2":
    print("Алу операциясы")
elif choice == "3":
    print("Бағдарлама аяқталды")
else:
    print("Белгісіз таңдау")

Мұндай программаларды меню негізіндегі (menu-driven) программалар деп атайды.

2-бөлім · Мысалдар

1-мысал · Саны оң, теріс немесе нөл

x = int(input("Санды енгіз: "))

if x > 0:
    print("Оң сан")
elif x < 0:
    print("Теріс сан")
else:
    print("Нөл")
Кіріс: -3
Шығыс: Теріс сан

2-мысал · Маусымды анықтау

Ай номері бойынша жыл мезгілін анықтаймыз.

month = int(input("Ай номерін енгіз (1-12): "))

if month == 12 or month == 1 or month == 2:
    print("Қыс")
elif month == 3 or month == 4 or month == 5:
    print("Көктем")
elif month == 6 or month == 7 or month == 8:
    print("Жаз")
elif month == 9 or month == 10 or month == 11:
    print("Күз")
else:
    print("Мұндай ай жоқ")

3-мысал · Қарапайым калькулятор (4 амал)

a = float(input("a = "))
b = float(input("b = "))
op = input("Амал таңдау (+, -, *, /): ")

if op == "+":
    print("Нәтиже:", a + b)
elif op == "-":
    print("Нәтиже:", a - b)
elif op == "*":
    print("Нәтиже:", a * b)
elif op == "/":
    if b != 0:
        print("Нәтиже:", a / b)
    else:
        print("Нөлге бөлуге болмайды!")
else:
    print("Белгісіз амал")

3-бөлім · Практика

Тапсырма 1 · Жасы бойынша категория

Пайдаланушының жасына қарай категорияны анықтайтын программа жаз:

  • 0–6 → "Мектепке дейінгі";
  • 7–17 → "Оқушы";
  • 18–22 → "Студент";
  • 23 және жоғары → "Ересек".
Кіріс (age) Шарт Шығыс
5 0 ≤ age ≤ 6 "Мектепке дейінгі"
15 7 ≤ age ≤ 17 "Оқушы"
20 18 ≤ age ≤ 22 "Студент"
30 age ≥ 23 "Ересек"

Тапсырма 2 · Балл бойынша баға

Пайдаланушыдан 0–100 аралығында балл енгізгізіп, келесі шкала бойынша баға шығаратын программа құрастыр:

  • 90–100 → "Өте жақсы";
  • 75–89 → "Жақсы";
  • 60–74 → "Қанағаттанарлық";
  • 0–59 → "Қанағаттанарлықсыз".
Кіріс (score) Шарт Шығыс
92 score ≥ 90 "Өте жақсы"
80 75 ≤ score < 90 "Жақсы"
65 60 ≤ score < 75 "Қанағаттанарлық"
40 score < 60 "Қанағаттанарлықсыз"

Тапсырма 3 · Күннің типін анықтау

Аптаның күнін (1–7) енгізгенде оның жұмыс күні немесе демалыс екенін шығаратын программа жаз:

  • 1–5 → "Жұмыс күні";
  • 6–7 → "Демалыс";
  • басқа мәндер → "Мұндай күн жоқ".
