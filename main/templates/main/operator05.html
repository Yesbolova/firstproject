Циклдегі else блогы

Бұл сабақта for және while циклдарындағы else блогының жұмыс принципін, оның break-пен байланысын және практикалық қолданылу мысалдарын қараймыз.

Оқу мақсаты

  • Циклден кейінгі else блогының мағынасын түсіну;
  • for ... else және while ... else конструкцияларын қолдануды үйрену;
  • break орындалғанда else-тің қалай жұмыс істейтінін түсіндіру;
  • Тізімнен элемент іздеу сияқты практикалық есептерді шешу;
  • Mini-quiz арқылы тақырыпты бекіту.

1-бөлім · Теория

Циклдегі else деген не?

Python тілінде for және while циклдарынан кейін else блогын жазуға болады. Ол:

  • цикл табиғи түрде аяқталса (яғни, break орындалмаса) – else іске қосылады;
  • цикл break арқылы ерте тоқтаса – else орындалмайды.

Яғни, else блогын «цикл сәтті аяқталды ма?» деп тексеретін қосымша блок ретінде қарастыруға болады.

for ... else жалпы түрі

for элемент in тізім:
    # қандай да бір әрекеттер
    if шарт:
        break
else:
    # егер break БОЛМАСА, осы жер орындалады

Мысал: тізімнен элемент іздеу

Тізімде берілген сан бар ма, жоқ па – соны for ... else арқылы тексерейік: 

numbers = [3, 8, 1, 10, 5]
target = 10

for n in numbers:
    if n == target:
        print("Мән табылды:", n)
        break
else:
    print("Мән тізімде жоқ")

Егер target тізімде болса, break орындалады да, else мүлде орындалмайды. Ал егер target табылмаса, цикл break-сіз аяқталады да, else блогы жұмыс істейді.

while ... else жалпы түрі

while шарт:
    # әрекеттер
    if басқа_шарт:
        break
else:
    # цикл break-ке жетпей аяқталса, орындалады

Мысал: бөлінетін сан табу

1-ден 20-ға дейінгі сандардың ішінде 7-ге бөлінетін сан бар ма?

i = 1

while i <= 20:
    if i % 7 == 0:
        print("7-ге бөлінетін сан табылды:", i)
        break
    i += 1
else:
    print("7-ге бөлінетін сан табылмады")

Егер цикл барысында 7-ге бөлінетін сан шықса – break арқылы тоқтайды және else орындалмайды. Ал егер шартқа сай сан мүлде табылмаса, цикл табиғи түрде аяқталып, else блогы іске қосылады.

2-бөлім · Мысалдар

1-мысал · Логин тексеру

Пайдаланушы дұрыс логин енгізді ме, соны тексереміз:

correct_users = ["admin", "python", "student"]
login = input("Логинді енгіз: ")

for user in correct_users:
    if login == user:
        print("Қош келдің,", login)
        break
else:
    print("Мұндай қолданушы табылмады")

2-мысал · Простой іздеу алгоритмі

Тізімдегі теріс санды іздеу:

nums = [4, 2, 7, 9, 3]
for x in nums:
    if x < 0:
        print("Теріс сан табылды:", x)
        break
else:
    print("Барлық сандар 0-ден үлкен немесе тең")

3-мысал · Простой пароль тексеру

3 ретке дейін пароль сұрап, бәрі қате болса, else арқылы хабар шығару:

password = "python123"

for attempt in range(3):
    guess = input("Құпия сөзді енгіз: ")
    if guess == password:
        print("Қош келдің!")
        break
    print("Қате, қайта енгіз.")
else:
    print("Кіру бұғатталды. Көп қате әрекет.")

3-бөлім · Практика

Тапсырма 1 · Тізімнен сан іздеу

Пайдаланушыдан сан енгізетін программа жаз. Тізім [2, 5, 7, 10, 15, 20] ішінде бұл сан бар болса – «табылды» деп жаз, болмаса – else арқылы «табылмады» деп жаз.

Тапсырма 2 · Жетілдірілген пароль

Құпия сөз "qwerty" болсын. Пайдаланушыға 5 ретке дейін пароль енгізуге мүмкіндік бер. Егер бір рет те дұрыс енгізбесе, else блогында «Барлық 5 әрекет қате болды» деген хабар шығар.

Тапсырма 3 · Бөлінетін санды іздеу

1-ден 50-ге дейінгі сандар арасынан 9-ға бөлінетін бірінші санды тап. Егер тапсаң – оны шығарып, break арқылы циклды тоқтат. Егер 9-ға бөлінетін сан болмаса (ойдан сценарий құр), else арқылы «9-ға бөлінетін сан жоқ» деген хабар бер.

1) Циклдегі else қашан орындалады?

2) Қай жазба дұрыс?

3) Төмендегі кодта else орындала ма?

for i in range(3):
    print(i)
else:
    print("Цикл аяқталды")

4) Төмендегі кодта «Табылмады» деген хабар қай кезде шығады?

for x in nums:
    if x == 10:
        print("Табылды")
        break
else:

5-бөлім · Үй тапсырмасы

Чек-лист

  • for ... else және while ... else синтаксисін білемін;
  • break болғанда else не істейтінін түсіндіре аламын;
  • Тізімнен элемент іздейтін программаны else-пен жаза аламын;
  • Mini-quiz сұрақтарына жауап бердім.

Үй тапсырмасы

  • Тізімдегі ең кіші санды табатын бағдарлама жаз. Егер тізім бос болса, else арқылы «Тізім бос» деп шығарып көр;
  • Пайдаланушыдан енгізілген сандардың ішінен тек жұп сандарды іздеп, егер бірде-бір жұп сан болмаса, else блогында хабар беретін программа жаз;
  • Кодты файлға сақтап, келесі сабақта көрсет.
