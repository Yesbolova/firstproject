Элементті жою және кірістіру

Оқу мақсаты: бір өлшемді, екі өлшемді массивтерді пайдаланып Python программалау тілінде программаларды жасау

Python-да бірөлшемді массивке элементтерді кірістіруді және өшіруді жүзеге асыру жолдарымен танысайық. Бұл әрекеттер Python тілінде өте оңай орындалады.

1-мысал (Элементтерді өшіру)

A[N] бірөлшемді массивінде нөлге тең жалғыз элемент бар. Сол элементті өшіріп, массивтің ұзындығын қысқарту керек. 


# 1-мысал. Нөлге тең элементті өшіру

# N (0 < N < 100) элемент саны
N = int(input("Элементтер санын енгізіңіз: "))

A = []
for i in range(N):
    x = int(input(f"{i + 1}-элементті енгізіңіз: "))
    A.append(x)

print("Бастапқы массив:", A)

# Массивтегі 0 тұрған орынды табамыз
zero_index = A.index(0)

# Сол элементті өшіреміз
A.pop(zero_index)

print("Нөлді өшіргеннен кейінгі массив:", A)
print("Жаңа ұзындығы:", len(A))

Нәтиже мысалы


A = [80, 11, 25, 14, 5, 43, 5, 0, 41, 75]
print("Бастапқы:", A)

zero_index = A.index(0)
A.pop(zero_index)

print("Нәтижесі:", A)
# Нәтиже: [80, 11, 25, 14, 5, 43, 5, 41, 75]

2-мысал (Элементтерді кірістіру)

B[N] массивінің K-шы орнына жаңа X элементін кірістіру керек. Python-да бұл insert() әдісімен орындалады. 


# 2-мысал. Элементті кірістіру

# N (0 < N < 100) элемент саны
N = int(input("Элементтер санын енгізіңіз: "))

B = []
for i in range(N):
    x = int(input(f"{i + 1}-элементті енгізіңіз: "))
    B.append(x)

print("Бастапқы массив:", B)

K = int(input("К-шы орнын енгізіңіз (1-ден басталады): "))
X = int(input("Жаңа элемент X мәнін енгізіңіз: "))

# Python индексі 0-ден басталады → сондықтан K-1
B.insert(K - 1, X)

print("Жаңа массив:", B)
print("Массив ұзындығы:", len(B))

Нәтиже мысалы


B = [80, 11, 25, 14, 5, 43, 5, 41, 75]
print("Бастапқы:", B)

K = 6
X = 55

B.insert(K - 1, X)

print("Нәтижесі:", B)
# Нәтиже: [80, 11, 25, 14, 5, 55, 43, 5, 41, 75]
