Жою және кірістіру (массив элементтерімен жұмыс)

Бұл сабақта бір өлшемді тізімдегі (массивтегі) элементтерді жою және жаңа элементтерді тізімнің басына, ортасына немесе соңына кірістіруді үйренеміз.

Сабақ соңында сен Python тізімдерімен еркін жұмыс жасап, сұрақтарға жауап берсең 100 XP жинай аласың.

Оқу мақсаты

  • Тізім (list) ұғымын және индекстерді қайталау;
  • Элементті индекс бойынша жою және мәні бойынша жою тәсілдерін түсіну;
  • append(), insert(), pop(), remove() әдістерін қолдану;
  • Тізімнің ортасына және кез келген орнына жаңа элементтерді кірістіру;
  • Mini-quiz арқылы тақырыпты бекіту.

1-бөлім · Теория

Тізім (list) және индекстер

Python тілінде бір өлшемді массивтің орнына көбіне тізім (list) қолданылады: 

a = [5, 9, 3, 7]
  • Элементтер: 5, 9, 3, 7;
  • Индекстер: a[0] – 5, a[1] – 9, a[2] – 3, a[3] – 7.

Элементті жою тәсілдері

1) Индекс арқылы жою: del және pop()

a = [5, 9, 3, 7]
del a[1]        # 1-индекстегі элементті жояды (9)
print(a)        # [5, 3, 7]
a = [10, 20, 30, 40]
x = a.pop(2)    # 2-индекстегі элементті жояды және мәнін қайтарады
print(x)        # 30
print(a)        # [10, 20, 40]
  • del a[i] – тек тізімнен жояды;
  • a.pop(i) – тізімнен жояды және жойылған мәнді қайтарады.

2) Мәні бойынша жою: remove()

a = [5, 9, 3, 9, 7]
a.remove(9)     # тізімдегі бірінші кездескен 9-ды жояды
print(a)        # [5, 3, 9, 7]

Егер мән тізімде жоқ болса, remove() қате береді.

Элементті кірістіру тәсілдері

1) Соңына қосу: append()

a = [2, 4, 6]
a.append(8)
print(a)        # [2, 4, 6, 8]

2) Кез келген позицияға кірістіру: insert()

a = [10, 20, 40]
a.insert(2, 30)     # 2-индекске 30 мәнін кірістіреді
print(a)            # [10, 20, 30, 40]
  • a.insert(0, x) – тізім басына кірістіреді;
  • a.insert(len(a), x) – тізім соңына қосу (append-ке ұқсас);
  • a.insert(k, x) – k-индекске қойып, қалғандарын оңға ығыстырады.

Бар элементті ауыстыру

Кейде элементті жоймай-ақ, тікелей мәнін өзгерту жеткілікті: 

a = [5, 9, 3]
a[1] = 10      # 9-дың орнына 10
print(a)       # [5, 10, 3]

Яғни жою – элементті мүлде алып тастау, ал ауыстыру – орнында қалдырып, мәнін өзгерту.

2-бөлім · Мысалдар

1-мысал · Белгілі индекстегі элементті жою

5-элементі бар тізімнен 2-индекстегі (үшінші) элементті өшіреміз.

nums = [4, 7, 9, 1, 5]
del nums[2]
print(nums)
# Консоль шығару:
[4, 7, 1, 5]

2-мысал · Барлық нөлдерді жою

Тізімдегі барлық 0 мәндерін жоямыз.

a = [0, 5, 0, 3, 0, 7]

while 0 in a:
    a.remove(0)

print(a)
# Консоль шығару:
[5, 3, 7]

3-мысал · Ортасына элемент кірістіру

Ортаға жаңа мән қосу:

a = [10, 20, 40, 50]

mid_index = len(a) // 2     # ортасы
a.insert(mid_index, 30)

print(a)
# Консоль шығару:
[10, 20, 30, 40, 50]

4-мысал · Емтихан балдарын өңдеу

Соңында жаңылыс енгізілген бағаны өшіріп, орнына дұрыс мәнді қой.

scores = [90, 75, 88, 40]   # 40 қате енгізілген

scores.pop()                # соңғы элементті жою
scores.append(98)           # дұрыс мәнді қосу

print(scores)
# Консоль шығару:
[90, 75, 88, 98]

3-бөлім · Практика

Тапсырма 1 · Тізімнен элемент жою

Пайдаланушыдан N және N бүтін сан енгізетін программа жаз. Кейін пайдаланушы енгізген k-индекстегі элементті жой.

Күтілетін кіріс/шығыс үлгісі:

Кіріс (input) Операция Шығыс (output)
n = 5, a = [3, 8, 1, 6, 4], k = 2 del a[2] [3, 8, 6, 4]

Тапсырма 2 · Тізім ортасына кірістіру

Тізім берілген. Пайдаланушыдан бір сан енгізіп, оны тізімнің дәл ортасына кірістіру программасын жаз.

Күтілетін кіріс/шығыс үлгісі:

Кіріс Операция Шығыс
a = [1, 2, 4, 5], x = 3 a.insert(2, 3) [1, 2, 3, 4, 5]

Тапсырма 3 · Теріс сандарды жою

Тізімдегі барлық теріс сандарды жойып, тек 0 және оң сандарды қалдыратын программа жаз.

Күтілетін кіріс/шығыс үлгісі:

Кіріс Логика Шығыс
[-3, 5, 0, -1, 8] теріс мәндерді өшіру [5, 0, 8]

3) Төмендегі кодтың нәтижесі қандай?

a = [1, 2, 3, 4]
a.insert(1, 9)
print(a)

4) remove(x) әдісі туралы қай тұжырым дұрыс?

5) Төмендегі код орындалғанда не шығады?

a = [5, 6, 7]
x = a.pop()
print(x)

6) Қай жазылу тізім басына 0 мәнін кірістіреді?

5-бөлім · Үй тапсырмасы

Үйде орындауға

  • 1-тапсырма: Ұзындығы кемінде 7 элемент болатын тізім құрып, оның ортасындағы 2 элементті жойып, орнына жаңа 2 элемент кірістір.
  • 2-тапсырма: Пайдаланушы енгізген тізімнен барлық 0 мәндерін жоятын программаны жаз.
  • 3-тапсырма: Өз ойыңнан мектеп журналындағы баға тізімін моделдейтін программа құрып, ондағы минимал бағаларды жойып, орнына жаңа, түзетілген бағаларды енгіз.
