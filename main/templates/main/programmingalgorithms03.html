Күрделі шарттарды программалау

Бұл сабақта Python тілінде күрделі шарттарды жазуды, and, or, not логикалық амалдарын қолдануды және логикалық өрнектердің орындалу ретін түсінеміз.

Сабақ соңында сен қарапайым шарттарды біріктіріп, күрделі логикалық өрнектер жаза аласың және Mini-Quiz арқылы біліміңді тексересің.

Оқу мақсаты

  • Қарапайым және күрделі шарт ұғымын ажырату;
  • and, or, not логикалық амалдарын түсіну;
  • Логикалық өрнектердің орындалу ретін білу;
  • Python тілінде күрделі шарттармен жұмыс істейтін бағдарламалар жазу;
  • Mini-quiz арқылы тақырыпты бекіту.

1-бөлім · Теория

Қарапайым және күрделі шарттар

Қарапайым шарт — бір ғана салыстырудан тұратын өрнек: a > b, x == 0, age >= 18 және т.б.

Кейде бір ғана шарт жеткіліксіз болады. Мысалы: «Күн суық және қар жауып тұр». Бұл жерде екі бөлек шарт бар:

  • күн суық
  • қар жауып тұр

Олар «ЖӘНЕ» сөзімен біріктірілген. Осындай бірнеше шарт бірігіп жазылған өрнек — күрделі шарт.

Күрделі шарт деген не?

Күрделі шарт — логикалық амалдар көмегімен біріктірілген екі немесе бірнеше қарапайым қатынастар (шарттар): 

(x > 0) and (x < 10)
age >= 18 or has_parent
not (x == 0)

Python-тағы логикалық амалдар

  • ЖӘНЕ — логикалық көбейту, Python-да and деп жазылады;
  • НЕМЕСЕ — логикалық қосу, Python-да or деп жазылады;
  • ЕМЕС — логикалық терістеу, Python-да not деп жазылады.

1) AND (ЖӘНЕ)

A and B өрнегі тек екі шарт та ақиқат болғанда ғана True болады. Егер бірінші шарт False болса, екінші шарт мүлдем тексерілмеуі де мүмкін. 

age = 15
has_ticket = True

can_enter = (age >= 14) and has_ticket
print(can_enter)  # True

2) OR (НЕМЕСЕ)

A or B өрнегі — кем дегенде бір шарт ақиқат болса, True. Тек екі шарт та жалған болғанда ғана False. 

is_student = True
has_discount_card = False

can_get_discount = is_student or has_discount_card
print(can_get_discount)  # True

3) NOT (ЕМЕС)

not — пікірді теріске шығарады: True → False, False → True. 

is_raining = False
go_for_a_walk = not is_raining
print(go_for_a_walk)  # True

Мысал: A < B және not (A >= B) — мағынасы бірдей шарттар.

Логикалық амалдар мен қатынастардың орындалу реті

Python-та күрделі өрнектерді есептегенде мына рет сақталады:

  1. Жақша ішіндегі амалдар;
  2. not амалы;
  3. Салыстыру амалдары: >, <, >=, <=, ==, !=;
  4. and амалы;
  5. or амалы.

Әрекеттердің ретін өзгерту үшін дөңгелек жақшаларды қолданамыз.

Логикалық айнымалылар

Логикалық айнымалы тек екі мәннің бірін қабылдайды: True (ақиқат) немесе False (жалған). 

x = 5
y = 10

is_less = x < y
is_equal = x == y

print(is_less)   # True
print(is_equal)  # False

2-бөлім · Мысалдар

1-мысал · Сан 1 мен 10 аралығында ма?

x = int(input("Санды енгіз: "))

if x >= 1 and x <= 10:
    print("Сан 1 мен 10 аралығында.")
else:
    print("Диапазоннан тыс.")
Кіріс: 7
Шығыс: Сан 1 мен 10 аралығында.

2-мысал · Емтиханнан өттің бе?

Егер баға 50-ден жоғары немесе қосымша жұмыс тапсырылса, студент өтеді.

score = int(input("Бағаңды енгіз: "))
extra_work = input("Қосымша жұмыс тапсырдың ба? (иә/жоқ): ")

has_extra = extra_work.lower() == "иә"

if score >= 50 or has_extra:
    print("Сен емтиханнан өттің!")
else:
    print("Әлі дайындалу керек.")

3-мысал · Тыйым салынған аймақ

Жас бойынша шектеу: 18 жастан кіші болса, кіргізілмейді.

age = int(input("Жасың: "))

if not (age >= 18):
    print("Кіруге рұқсат жоқ (жасың 18-ге толмаған).")
else:
    print("Кіруге рұқсат.")

4-мысал · Қауіпсіз пароль ме?

Қарапайым логика: пароль қысқа немесе бос болса — жарамсыз.

password = input("Пароль енгіз: ")

if password == "" or len(password) < 6:
    print("Пароль тым қысқа немесе бос.")
else:
    print("Пароль қабылданды.")

3-бөлім · Практика

Тапсырма 1 · Сан диапазонда ма?

Пайдаланушыдан бір бүтін сан енгізіп, оның 1 мен 100 аралығында екенін және 5-ке бөлінбейтінін тексеретін программа жаз.

Кіріс (x) Шарт Шығыс
20 1 ≤ x ≤ 100 and x % 5 != 0 False (5-ке бөлінеді)
37 1 ≤ x ≤ 100 and x % 5 != 0 True

Тапсырма 2 · Күн райына кеңес

Пайдаланушыдан ауа температурасы мен жауын-шашын туралы ақпарат ал: t — температура, rain — «иә/жоқ». Келесі логика бойынша хабарлама шығаратын программа жаз:

  • Егер t < 0 және rain == "иә" → "Қар аралас жаңбыр, жылы киін!";
  • Егер t >= 20 немесе жаңбыр емес → "Жылы ауа райы.";
  • Қалған жағдайда → "Ауа райы өзгермелі.".

Тапсырма 3 · Қарапайым авторизация

Логин мен пароль бойынша кіргізетін программа жаз:

  • Логин "student", пароль ұзындығы кемінде 6 символ;
  • Егер логин дұрыс және пароль ұзындығы жеткілікті болса → "Қош келдің, студент!";
  • Әйтпесе → "Логин немесе пароль қате.".
