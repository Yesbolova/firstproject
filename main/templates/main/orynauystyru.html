Элементтерді ауыстыру

Бұл сабақта бір өлшемді массивтегі элементтердің орындарын ауыстыру, уақытша айнымалыны пайдалану және Python тілінде массивпен (тізіммен) жұмыс істеуді қайталаймыз.

Оқу мақсаты

  • Бір өлшемді массивтегі элементтерді индекстер бойынша түсіну;
  • Екі элементтің орнын ауыстыру алгоритмін жазу;
  • Уақытша айнымалы (temp) қолдану идеясын меңгеру;
  • Python тілінде тізім элементтерін ауыстырудың қысқа жазылуын қолдану;
  • Практикалық есептер арқылы дағдыны бекіту.

1-бөлім · Теория

Бір өлшемді массив және индекстер

Python тілінде бір өлшемді массивті көбіне тізім (list) арқылы көрсетеміз: 

a = [5, 9, 2, 7, 3]

Мұнда a — тізім, ал әр элементтің индексі болады:

  • a[0] = 5
  • a[1] = 9
  • a[2] = 2 және т.с.с.

Python-да индекстер 0-ден басталады. Сондықтан бірінші элемент — a[0], ал соңғысы — a[len(a) - 1].

Екі элементтің орнын қалай ауыстырамыз?

Классикалық тәсіл — уақытша айнымалы пайдалану:

a = [5, 9, 2, 7, 3]

# 1 және 3-индекстегі элементтердің орнын ауыстыру (9 және 7)
temp = a[1]
a[1] = a[3]
a[3] = temp

print(a)

Бағдарлама нәтижесі:

[5, 7, 2, 9, 3]

Яғни, a[1] мен a[3] мәндері бір-бірімен орын ауыстырды.

Python-дағы қысқа жазылу

Python тілінде екі айнымалының орнын уақытша айнымалысыз-ақ бір жолда ауыстыруға болады: 

a[i], a[j] = a[j], a[i]

Мысал:

a = [5, 9, 2, 7, 3]

i = 1
j = 3
a[i], a[j] = a[j], a[i]

print(a)

Нәтиже:

[5, 7, 2, 9, 3]

Бастан-аяқ алгоритм (алғашқы және соңғы элемент)

Массивтің бірінші және соңғы элементтерінің орнын ауыстыру үшін мынадай алгоритм қолданамыз:

  1. Тізімді енгізу (немесе дайын тізім беру);
  2. Бірінші элемент индексі — 0, соңғы элемент индексі — len(a) - 1;
  3. Екі элементтің орнын ауыстыру;
  4. Жаңартылған тізімді шығару.
a = [10, 20, 30, 40, 50]

first = 0
last = len(a) - 1

a[first], a[last] = a[last], a[first]

print(a)

Бағдарлама нәтижесі:

[50, 20, 30, 40, 10]

2-бөлім · Мысалдар

1-мысал · Бірінші және соңғы элементті ауыстыру

a = [4, 8, 15, 16, 23, 42]

first = 0
last = len(a) - 1

a[first], a[last] = a[last], a[first]

print("Жаңартылған тізім:", a)

Консолдағы нәтиже:

Жаңартылған тізім: [42, 8, 15, 16, 23, 4]

2-мысал · Екі көрсетілген индексті ауыстыру

Пайдаланушыдан индекстерді енгіздіреміз:

a = [5, 9, 2, 7, 3, 1]

print("Бастапқы тізім:", a)

i = int(input("Бірінші индексті енгіз: "))
j = int(input("Екінші индексті енгіз: "))

a[i], a[j] = a[j], a[i]

print("Жаңартылған тізім:", a)

Мысал жұмыс (input/output):

Бастапқы тізім: [5, 9, 2, 7, 3, 1]
Бірінші индексті енгіз: 1
Екінші индексті енгіз: 4
Жаңартылған тізім: [5, 3, 2, 7, 9, 1]

3-мысал · Тізімнің ортасын айна секілді ауыстыру

Бірінші элемент соңғысымен, екінші элемент соңғыдан біріншісімен, т.с.с.

a = [1, 2, 3, 4, 5, 6]

left = 0
right = len(a) - 1

while left < right:
    a[left], a[right] = a[right], a[left]
    left += 1
    right -= 1

print("Айналған тізім:", a)

Нәтиже:

Айналған тізім: [6, 5, 4, 3, 2, 1]

3-бөлім · Практика

Тапсырма 1 · Екі элементтің орнын ауыстыр

Тізім берілсін: a = [10, 20, 30, 40, 50]. Пайдаланушыдан екі индекс енгізіп, сол индекстердегі элементтердің орнын ауыстыратын программа жаз.

Мысал кіріс және шығыс:

Кіріс Орындалған әрекет Шығыс
i = 1, j = 3 [10, 20, 30, 40, 50] → a[1] ↔ a[3] [10, 40, 30, 20, 50]

Тапсырма 2 · Максимумды бірінші орынға әкел

Берілген тізімдегі ең үлкен элементті тауып, оны бірінші элементпен орын ауыстыр: 

a = [3, 11, 7, 5, 9]

Қадамдық ойлау:

  • Максимум мәнін және оның индексін тап;
  • Бірінші элементпен (a[0]) орнын ауыстыр;
  • Нәтижені тізім түрінде шығар.

Тапсырма 3 · Көрші элементтердің орнын ауыстыр

Тізімдегі (0,1), (2,3), (4,5) т.с.с. жұп индекстерді өзара ауыстыр: 

a = [1, 2, 3, 4, 5, 6]

Күтілетін логика:

Бастапқы тізім Ауыстырулар Жаңартылған тізім
[1, 2, 3, 4, 5, 6] a[0]↔a[1], a[2]↔a[3], a[4]↔a[5] [2, 1, 4, 3, 6, 5]
