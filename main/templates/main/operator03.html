Continue операторы

Бұл сабақта біз continue операторының не үшін қолданылатынын, ол циклдың жұмысын қалай өзгертетінін, break операторынан айырмашылығын және практикалық мысалдарын қарастырамыз.

Оқу мақсаты

  • continue операторының мағынасын түсіну;
  • for және while циклдарында continue қолдануды үйрену;
  • break, continue, pass операторларын салыстыру;
  • Практикалық есептер арқылы шартты өткізіп кетуді қолдану;
  • Mini-quiz арқылы білімді тексеру.

1-бөлім · Теория

Continue операторы деген не?

continue операторы — цикл ішінде белгілі бір жағдай орындалса, ағымдағы итерацияның қалған бөлігін өткізіп жіберіп, бірден келесі итерацияға өтуді қамтамасыз ететін оператор.

Яғни:

  • цикл тоқтамайды;
  • тек сол сәттегі итерацияның қалған командалары орындалмайды;
  • келесі қайталануға өтіп кетеді.

break, continue, pass айырмашылығы

  • break — циклды толық тоқтатады;
  • continue — тек ағымдағы итерацияны өткізіп кетеді, цикл жалғасады;
  • pass — «ештеңе істеме» деген бос оператор, кодтың орнын «толтыру» үшін.

Continue операторы бар for циклі

1-ден 10-ға дейінгі сандарды шығарып, тек жұп сандарды ғана көрсетейік:

for i in range(1, 11):
    if i % 2 != 0:
        continue      # тақ сандарды өткізіп жібереміз
    print(i)

2
4
6
8
10

Мұнда i тақ болғанда, continue орындалады да, сол итерациядағы print(i) командасы орындалмайды.

Continue операторы бар while циклі

1-ден 5-ке дейінгі сандарды шығарайық, бірақ 3 санын өткізіп жібереміз:

i = 1

while i <= 5:
    if i == 3:
        i += 1       # міндетті түрде i-ді үлкейтеміз
        continue     # 3-ті өткізіп кетеміз

    print(i)
    i += 1

Нәтиже:

1
2
4
5

Назар аудар: while циклында continue алдында i += 1 жазбасақ, шексіз цикл туындайды (өйткені i мәні өзгермейді).

Жол ішіндегі кейбір таңбаларды өткізіп кету

Берілген мәтіндегі бос орындарды өткізіп, тек символдарды шығарайық:

text = "Hello Python"

for ch in text:
    if ch == " ":
        continue      # бос орындарды өткіземіз
    print(ch)

Нәтиже: әр әріп жеке жолға жазылады, бірақ бос орындар жоқ.

2-бөлім · Мысалдар

1-мысал · Теріс сандарды өткізіп кету

Тізімдегі сандардың тек оң мәндерін ғана шығарайық:

numbers = [3, -5, 7, 0, -2, 10]

for n in numbers:
    if n <= 0:
        continue      # теріс және 0 сандарын өткізіп жібереміз
    print(n)

Нәтиже:

3
7
10

2-мысал · Тек дауысты әріптерді шығару

Мәтіндегі тек дауысты әріптерді шығарайық:

text = "informatika"
vowels = "aeiou"

for ch in text:
    if ch not in vowels:
        continue      # дауыссыздарды өткізіп жібереміз
    print(ch)

Нәтиже:

i
o
a
i
a

3-мысал · 1–20 аралығындағы 3-ке бөлінбейтін сандар

1 ден 20-ға дейінгі сандардан 3-ке бөлінетіндерін өткізіп жіберейік:

for i in range(1, 21):
    if i % 3 == 0:
        continue
    print(i)

3-бөлім · Практика

Тапсырма 1 · Жұп сандарды өткізіп жібер

1-ден 20-ға дейінгі сандардың ішінен тек тақ сандарды шығаратын программа жаз. Жұп сандарды continue арқылы өткізіп жібер.

Күтілетін нәтиже (фрагмент):

3
5
...
19

Тапсырма 2 · Теріс мәндерді елемеу

Пайдаланушыдан бірнеше сан енгізілген тізім ал. Цикл ішінде теріс сандарды continue арқылы өткізіп, тек нөлден үлкен сандардың қосындысын шығар.

Мысал:

Кіріс: [5, -2, 3, -1, 4]
Шығыс: 12

Тапсырма 3 · Құпия сөзді тексеру

Пайдаланушыдан бірнеше рет сөз енгізетін программа жаз. Егер енгізілген жол бос болса (""), оны continue арқылы өткізіп жібер. Тек бос емес жолдарды ғана «қабылданды» деп шығарып отыр.

4-бөлім · Mini-Quiz

Төмендегі тест сұрақтарына жауап бер. Соңында «Тексеру» батырмасын бас.

1) Continue операторы не істейді?

2) Қай оператор циклды толық тоқтатады?

3) Төмендегі код не істейді?

for i in range(5):
    if i == 2:
        continue
    print(i)

4) Қай жағдайда continue қолдану ыңғайлы?

5) While циклында continue қолданғанда нені естен шығармау керек?

5-бөлім · Үй тапсырмасы

Чек-лист

  • continue операторын түсіндіре аламын;
  • break, continue және pass айырмашылығын айта аламын;
  • for және while циклдарында continue қолданып программа жаза аламын;
  • Mini-quiz сұрақтарына жауап бердім.

Үй тапсырмасы

  • Тізімдегі сандардың ішінен тек жұп сандардың қосындысын есептейтін программа жаз (тақ сандарды continue арқылы өткізіп жібер);
  • Мәтіндегі тек қазақ дауысты әріптерін (a, ә, e, i, о, ө, ұ, ү, ы, і) шығаратын программа жаз;
  • Бағдарламаларыңды файлға сақтап, келесі сабақта көрсет.
