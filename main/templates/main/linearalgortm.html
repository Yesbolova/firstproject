Сызықтық алгоритмдерді программалау

Бұл сабақта сызықтық алгоритм ұғымын қайталаймыз, математикалық өрнектерді Python тілінде жазып үйренеміз және бірнеше мысалды практикада орындаймыз.

Сабақты толық орындасаң, шамамен 100 XP жинайсың.

Оқу мақсаты

  • Сызықтық алгоритмнің анықтамасын түсіндіре алу;
  • Математикалық өрнектерді Python тілінде жазу;
  • Айнымалыға мән беріп, аралық нәтижелерді есептеу;
  • input() және print() көмегімен программа құру;
  • Mini-Quiz арқылы өз біліміңді тексеру.

1-бөлім · Сызықтық алгоритм деген не?

Сызықтық алгоритм – іс-әрекеттер (командалар) бірінен кейін бірі рет-ретімен орындалатын алгоритм. Яғни тармақталу (if) немесе цикл (for, while) жоқ, тек тізбекті командалар тізімі.

Мысалы, қарапайым сызықтық алгоритм:

  1. Пайдаланушыдан b санын енгізу;
  2. Берілген формула бойынша мән есептеу;
  3. Нәтижені экранға шығару.

Python тіліндегі жалпы үлгісі:

# 1-қадам: деректерді енгізу
b = float(input("b-ға мән бер: "))

# 2-қадам: есептеу
a = 1/9 * b**3 - 6.4 * (b + 8) + 3/4 * b

# 3-қадам: нәтижені шығару
print("Нәтиже:", a)

Командалар жоғарыдан төмен қарай біртіндеп орындалады – бұл сызықтық алгоритмнің негізгі белгісі.

Есептің формулалары

Төменде сызықтық алгоритмдерге мысал болатын өрнектер берілген:

  1. \( a = \frac{1}{9} b^3 - 6.4(b + 8) + \frac{3}{4}b \)
  2. \( d = \frac{1}{7} a^3 b - 2ab^2(a + b) \)
  3. \( q = \frac{3a^3 - 2ab + b^2}{2a(3a - b)} \)
  4. \( k = x^3 - 4xy(x + y^2) \)
  5. \( h = |v + 1.7| \cdot (2v + 3) \)

Осы формулаларды Python тілінде жазып, әрқайсысы үшін жеке программа құрамыз.

2-бөлім · Мысалдар

1-мысал · a шамасын есептеу

Берілген формула бойынша a мәнін табу үшін пайдаланушыдан b санын енгіземіз: 

b = float(input("b-ға мән бер: "))

a = 1/9 * b**3 - 6.4 * (b + 8) + 3/4 * b

print("Нәтиже:", a)

Консольдағы нәтиже (мысал):

b-ға мән бер: 3
Нәтиже: -32.95

2-мысал · d шамасын есептеу

Пайдаланушыдан a және b мәндерін енгізіп, d шамасын есептейміз: 

a = float(input("a-ға мән бер: "))
b = float(input("b-ға мән бер: "))

d = 1/7 * a**3 * b - 2 * a * b**2 * (a + b)

print("Нәтиже:", d)

Консольдағы нәтиже (мысал):

a-ға мән бер: 2
b-ға мән бер: 3
Нәтиже: -108.0

3-мысал · q шамасын есептеу

a және b мәндері бойынша бөлшек түріндегі өрнектің нәтижесін табамыз: 

a = float(input("a-ға мән бер: "))
b = float(input("b-ға мән бер: "))

q = (3 * a**3 - 2 * a * b + b**2) / (2 * a * (3 * a - b))

print("Нәтиже:", q)

Консольдағы нәтиже (мысал):

a-ға мән бер: 4
b-ға мән бер: 2
Нәтиже: 1.2916666667

4-мысал · k шамасын есептеу

Екі айнымалысы бар өрнек: k = x³ − 4xy(x + y²). 

x = float(input("x-ке мән бер: "))
y = float(input("y-ке мән бер: "))

k = x**3 - 4 * x * y * (x + y**2)

print("Нәтиже:", k)

Консольдағы нәтиже (мысал):

x-ке мән бер: 2
y-ке мән бер: 1
Нәтиже: -10

5-мысал · h шамасын есептеу

h = |v + 1.7| · (2v + 3) өрнегін есептейміз. Модульді есептеу үшін abs() функциясын қолданамыз. 

v = float(input("v-ге мән бер: "))

h = abs(v + 1.7) * (2 * v + 3)

print("Нәтиже:", h)

Консольдағы нәтиже (мысал):

v-ге мән бер: -2
Нәтиже: 0.8999999999

3-бөлім · Практика

Тапсырма 1

Жоғарыдағы 1–5 мысалдың бірін таңдап, программаға әртүрлі мәндер енгізіп көр. Нәтиженің қалай өзгеретінін бақыла.

Кіріс/шығыс деректер кестесі

Мысал Кіріс деректер (input) Шығыс дерек (output)
1-мысал b — нақты сан a — формула нәтижесі
2-мысал a, b — нақты сандар d — формула нәтижесі
3-мысал a, b — нақты сандар q — бөлшек өрнек нәтижесі
4-мысал x, y — нақты сандар k — өрнек нәтижесі
5-мысал v — нақты сан h — модуль бар өрнек нәтижесі

Тапсырма 2

Өз формулаңды ойлап тап (мысалы, үшбұрыш периметрі, тікбұрыш ауданын табу) және оны сызықтық алгоритм ретінде Python тілінде жаз.

Кіріс/шығыс деректерін жоспарлау

Формула / есеп Кіріс деректер (input) Шығыс дерек (output)
Үшбұрыш периметрі a, b, c — қабырғалар P — периметр
Тікбұрыш ауданы a, b — қабырғалар S — аудан
Өз есебі (оқушы) Айнымалылар тізімі Нәтиже атауы

Тапсырма 3

Бір программа ішінде қатарынан 3 түрлі өрнек есептейтін сызықтық алгоритм құр. Әр формула үшін бөлек айнымалы пайдалан.

Бағдарлама құрылымын жоспарлау

Қадам Кіріс деректер Шығыс / айнымалы
1-өрнек Бастапқы айнымалылар res1
2-өрнек Қажет болса res1 және басқа айнымалылар res2
3-өрнек Алдыңғы нәтижелер мен айнымалылар res3
