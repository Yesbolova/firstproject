Break операторы

Бұл сабақта біз break операторының не істейтінін, циклды қалай тоқтататынын, continue және else-пен байланысын, сондай-ақ практикалық мысалдар мен тест тапсырмаларын қарастырамыз.

Оқу мақсаты

  • break операторының жұмысын түсіну;
  • for және while циклдарында break қолдануды үйрену;
  • break пен continue айырмашылығын түсіндіре алу;
  • Циклдан ерте шығу қажет болатын практикалық есептерді шешу;
  • Mini-quiz арқылы өз біліміңді тексеру.

1-бөлім · Теория

Break операторы деген не?

break операторы цикл ішінде белгілі бір шарт орындалғанда цикл жұмысын толық тоқтату үшін қолданылады. Яғни:

  • ағымдағы итерация да, қалған итерациялар да жалғаспайды;
  • бағдарлама циклдан кейінгі бірінші командаларға өтеді.

break vs continue

  • break — циклды біржола аяқтайды;
  • continue — тек ағымдағы итерацияны өткізіп, келесі итерацияға өтеді;
  • pass — ештеңе істемейді (бос оператор).

Break операторы бар for циклі

0-ден бастап сандарды шығарып, 5-ке жеткенде циклды тоқтатайық:

for i in range(10):
    if i == 5:
        break     # 5-ке жеткенде цикл аяқталады
    print(i)

Break операторы бар while циклі

Пайдаланушыдан сандар енгізіп, 0 енгізгенде циклды тоқтатайық:

total = 0

while True:  # шексіз цикл
    x = int(input("Сан енгіз (тоқтату үшін 0): "))
    if x == 0:
        break          # 0 болса, цикл аяқталады
    total += x

print("Қосынды:", total)

Мұнда while True — шексіз цикл, ал break оны пайдаланушы 0 енгізгенде тоқтатады.

for ... else және break

Python-да циклдан кейін else блогын жазуға болады. break орындалмаса, else бөлімі іске қосылады; ал егер break арқылы цикл ерте тоқтаса, else орындалмайды. 

for i in range(1, 6):
    if i == 3:
        print("3 табылды!")
        break
    print("Тексеру:", i)
else:
    print("3 табылмады")

Бұл мысалда 3 табылғандықтан, break орындалады, цикл тоқтайды және else мүлде орындалмайды.

2-бөлім · Мысалдар

1-мысал · Құпия санды табу

Компьютер бір құпия сан «ойлап» тұр, ал біз оны табуымыз керек:

secret = 7

while True:
    guess = int(input("Болжам енгіз: "))
    if guess == secret:
        print("Дұрыс таптың!")
        break
    print("Қайтадан байқап көр!")

2-мысал · Тізімнен белгілі элементті іздеу

Тізімде қандай да бір сан бар ма, соны тексерейік:

numbers = [3, 8, 1, 10, 5]
target = 10

found = False

for n in numbers:
    if n == target:
        found = True
        break

if found:
    print("Мән тізімде бар!")
else:
    print("Мән табылмады.")

3-мысал · Теріс сан кездескенше қосу

Тізімдегі сандарды қосамыз, бірақ бірінші теріс санға дейін ғана:

nums = [5, 4, 3, -1, 10, 2]
total = 0

for x in nums:
    if x < 0:
        break      # теріс сан шыққан бойда тоқтаймыз
    total += x

print("Қосынды:", total)

3-бөлім · Практика

Тапсырма 1 · Теріс санға дейінгі қосынды

Пайдаланушыдан сандарды бір-бірлеп енгізетін программа жаз. Егер теріс сан енгізсе, цикл break арқылы тоқтайды, ал теріс санға дейін енгізілген сандардың қосындысы шығарылады.

Мысал:

Кіріс: 5, 3, 2, -1
Шығыс: 10

Тапсырма 2 · Құпия сөз

Құпия сөз алдын ала "python" деп берілсін. Пайдаланушыдан бірнеше рет пароль сұрайтын программа жаз. Егер пароль дұрыс болса, "Қош келдің!" деп жазып, циклды break арқылы тоқтат. Егер 3 рет қате енгізсе, "Кіру бұғатталды" деп шығарып, циклды тоқтат.

Тапсырма 3 · 1–100 аралығында іздеу

1-ден 100-ге дейінгі сандардан 25-ке бірінші бөлінетін санды тауып, оны экранға шығар да, циклды break арқылы тоқтат.

5-бөлім · Үй тапсырмасы

Чек-лист

  • break операторын анық түсіндіре аламын;
  • break пен continue айырмашылығын білемін;
  • Циклдан ерте шығу керек есептерді шеше аламын;
  • Mini-quiz сұрақтарына жауап бердім.

Үй тапсырмасы

  • Пайдаланушыдан сандар енгізіп, бірінші 0 мәні шыққанға дейін қосындысын табатын бағдарлама жаз;
  • Логин мен парольді 3 ретке дейін сұрап, дұрыс енгізгенде «Қош келдің!» деп, қате болса «Кіру сәтсіз аяқталды» деп шығаратын программа жаз (циклды break арқылы тоқтат);
  • Бағдарламаларыңды файлға сақтап, келесі сабақта көрсет.
