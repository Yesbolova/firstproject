While циклі

Бұл сабақта while циклі қалай жұмыс істейтінін, қай кезде қолдану тиімді екенін, санау, қосынды табу және қарапайым есептерді қайталап орындау үшін while-ды пайдалануды үйренеміз.

Сабақ соңында сен while циклін түсініп, бірнеше практикалық есеп шығарып, 10 сұрақтық Mini-Quiz арқылы біліміңді тексере аласың.

Оқу мақсаты

  • while циклі не үшін керектігін түсіну;
  • Қарапайым санау циклін жасау (1-ден n-ге дейін, кері санау);
  • Қайталанатын әрекеттер арқылы қосынды/көбейтінді табу;
  • Шарт жалған болғанда циклдің қалай тоқтайтынын түсіну;
  • Mini-quiz арқылы while тақырыбын бекіту.

1-бөлім · Теория

While циклі деген не?

Кейде бір әрекетті бірнеше рет қайталау керек болады: мысалы, 1-ден 10-ға дейін сандарды шығару, кіріс деректерді бірнеше рет сұрау, қосындыны есептеу т.б.

Осындай қайталанатын әрекеттерді орындау үшін циклдер қолданылады. Python тілінде циклдердің бірі — while циклі.

while циклі жалпы түрде былай жазылады:

while шарт:
    # шарт ақиқат (True) болғанша қайталанатын командалар
    команда1
    команда2
    ...
  • шарт – логикалық өрнек (True/False);
  • шарт ақиқат (True) болса – цикл ішіндегі блок орындала береді;
  • шарт жалған (False) болғанда – цикл тоқтайды.

1-ден 5-ке дейін сандарды шығару

i = 1

while i <= 5:
    print(i)
    i = i + 1
Кіріс: (жоқ)
Шығыс:
1
2
3
4
5

Мұнда i әр қайталаудан кейін 1-ге артып отырады. i <= 5 шарты жалған болған кезде (i = 6) цикл тоқтайды.

Кері санау мысалы

i = 5

while i >= 1:
    print(i)
    i = i - 1

print("Старт!")
Шығыс:
5
4
3
2
1
Старт!

Қайталанатын қосынды

1-ден n-ге дейінгі сандардың қосындысын табу:

n = int(input("n санын енгіз: "))

s = 0      # қосынды
i = 1      # есептегіш

while i <= n:
    s = s + i
    i = i + 1

print("Қосынды:", s)
Кіріс: n = 5
Шығыс: Қосынды: 15

2-бөлім · Мысалдар

1-мысал · Тек жұп сандарды шығару

1-ден n-ге дейінгі жұп сандарды шығаратын программа:

n = int(input("n санын енгіз: "))

i = 1
while i <= n:
    if i % 2 == 0:
        print(i)
    i = i + 1
Кіріс: n = 7
Шығыс:
2
4
6

2-мысал · Цифрлар қосындысы

Берілген санның цифрларының қосындысын табу:

n = int(input("Санды енгіз: "))

s = 0
while n > 0:
    digit = n % 10     # соңғы цифр
    s = s + digit      # қосамыз
    n = n // 10        # соңғы цифрды алып тастаймыз

print("Цифрлар қосындысы:", s)

3-мысал · Пайдаланушы 0 енгізгенше

Пайдаланушы 0 енгізгенше сандарды қосатын программа:

s = 0

x = int(input("Сан енгіз (тоқтау үшін 0): "))

while x != 0:
    s = s + x
    x = int(input("Тағы сан енгіз (тоқтау үшін 0): "))

print("Қосынды:", s)

Тапсырма 3 · Қарапайым «сан ойлау» ойыны

1–10 аралығында бір құпия сан ойлап, оқушыдан оны табуды сұрайтын программа жаз. Пайдаланушы дұрыс тапқанға дейін while циклі жалғассын.
