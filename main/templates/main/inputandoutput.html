Сандарды енгізу және шығару

Бұл сабақта Python тілінде print() арқылы сандарды экранға шығару, input() арқылы пайдаланушыдан деректерді енгізу және кейбір математикалық функцияларды қолдануды үйренеміз.

Сабақты толық аяқтағанда шамамен 100 XP жинайсың.

Дескриптор

  • print() функциясының жұмысын түсіну;
  • input() арқылы деректерді енгізуді меңгеру;
  • Енгізілген мәтінді санға түрлендіруді үйрену;
  • math модуліндегі кейбір функциялармен танысу;
  • Mini-Quiz арқылы тақырыпты бекіту.

1-бөлім · Сандарды шығару (print)

print() функциясы

Python-да біз шығыс деректерін экранға шығару үшін print() функциясын қолданамыз. Жолды тырнақша ішінде жазамыз. 

print("Python өте мықты тіл")

Бұл код экранда Python өте мықты тіл деген жазуды көрсетеді.

Бірнеше мәнді шығару

print() ішінде бірнеше мәнді үтір арқылы бөлуге болады. Олар бір жолда бос орынмен бөлініп шығады. 

age = 13
print("Менің жасым:", age)

2-бөлім · Сандарды енгізу (input)

input() функциясы

Программалау кезінде біз пайдаланушыдан деректер алғымыз келеді. Ол үшін input() функциясы қолданылады. Бұл функция пернетақтадан мән енгізуді күтеді. 

name = input("Атыңызды енгізіңіз: ")
print("Сәлем,", name)

Мұнда экранда «Атыңызды енгізіңіз: » деген жазу шығады, пайдаланушы атының орнына кез келген мәтін жазады, кейін бағдарлама «Сәлем, <аты>» деп жауап береді.

Мәтінді санға түрлендіру

input() қайтарған мән әрқашан str (жол) түрінде болады. Егер бізге сан керек болса, оны int() немесе float() арқылы түрлендіреміз. 

num = int(input("Санды енгізіңіз: "))

print("Сіз енгізген сан:", num)
print("Санның деректер типі:", type(num))

3-бөлім · math модулі

Математикалық басқа функциялар math модуліне біріктірілген. Python-дағы модуль – ішінде дайын функциялар бар файл. Математикалық функцияларды қолдану үшін модулді import командасы арқылы қосамыз. 

import math

# Кейбір негізгі функциялар:
# math.pow(num, power)  – num санын power дәрежесіне шығарады
# math.sqrt(num)        – num санының квадрат түбірін табады
# math.ceil(num)        – нақты санды жоғары жақтағы бүтінге дөңгелектейді
# math.floor(num)       – нақты санды төмен жақтағы бүтінге дөңгелектейді

x = 5
print(math.pow(x, 2))   # 25.0
print(math.sqrt(16))    # 4.0

4-бөлім · Практика

Тапсырма 1 · Атыңды шығару

Пайдаланушыдан атын сұрап, «Сәлем, <аты>!» деп шығаратын бағдарлама жаз.

Тапсырма 2 · Екі санның қосындысы

Пайдаланушыдан екі бүтін сан сұра, оларды int() арқылы санға айналдырып, қосындысын шығар: 

a = int(input("Бірінші сан: "))
b = int(input("Екінші сан: "))

print("Қосынды:", a + b)

Тапсырма 3 · Шаршы ауданы

Қабырғасы a болатын шаршының ауданын есептейтін бағдарлама жаз (ауданы: S = a * a). Қабырғаны пайдаланушыдан енгізуді сұра.

Тапсырма 4 · math модулін қолдану

Пайдаланушы енгізген санның квадрат түбірін табатын бағдарлама құр. 

import math

num = float(input("Сан енгізіңіз: "))
root = math.sqrt(num)

print("Квадрат түбірі:", root)

6-бөлім · Қорытынды және ресурстар

Есте сақта
