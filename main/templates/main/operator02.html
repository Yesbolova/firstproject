For циклі

Бұл сабақта for циклі қалай жазылатынын, range() функциясымен бірге қалай қолданылатынын, тізім элементтерін, жолдағы символдарды және сандарды қайталап өңдеуді үйренеміз.

Сабақ соңында сен for циклін түрлі жағдайларда пайдалана аласың және 10 сұрақтық Mini-Quiz арқылы біліміңді тексересің.

Оқу мақсаты

  • for циклі мен while циклінің айырмашылығын түсіну;
  • range() функциясының жұмысын меңгеру;
  • Сандар аралығын, тізімді, жолды for арқылы қайталап шығу;
  • Қарапайым есептерді for циклімен шешу;
  • Mini-quiz арқылы білімді бекіту.

1-бөлім · Теория

For циклі деген не?

For циклі – белгілі бір элементтер жиынын (мысалы, сандар аралығы, тізім, жолдағы символдар) ретімен өтіп, әр элемент үшін бірдей әрекет орындауға арналған цикл.

Жалпы жазылуы:

for айнымалы in реттік_жиын:
    # әр элемент үшін қайталанатын командалар
    команда1
    команда2
    ...
  • айнымалы – әр қадамда жаңа мән алып отырады;
  • реттік_жиын – range(), тізім, жол және т.б.;
  • цикл барлық элементтер біткенше қайталанады.

Range() функциясы

range() – сандар тізбегін құратын функция.

range(stop)          # 0-ден stop-1 дейін
range(start, stop)   # start-тан stop-1 дейін
range(start, stop, step)  # step қадамымен

Мысал: 0-ден 4-ке дейін

for i in range(5):
    print(i)
Шығыс:
0
1
2
3
4

Мысал: 1-ден 5-ке дейін

for i in range(1, 6):
    print(i)
Шығыс:
1
2
3
4
5

Мысал: 2-ден 10-ға дейін 2 қадаммен

for i in range(2, 11, 2):
    print(i)
Шығыс:
2
4
6
8
10

Тізім және жол бойынша цикл

Тізім элементтері бойынша өту:

languages = ["Python", "Java", "C++"]

for lang in languages:
    print(lang)
Шығыс:
Python
Java
C++

Жол символдары бойынша өту:

text = "Salem"

for ch in text:
    print(ch)

2-бөлім · Мысалдар

1-мысал · 1-ден n-ге дейінгі қосынды

n = int(input("n санын енгіз: "))

s = 0
for i in range(1, n + 1):
    s = s + i

print("Қосынды:", s)
Кіріс: n = 5
Шығыс: Қосынды: 15

2-мысал · Квадраттарды шығару

1-ден n-ге дейінгі сандардың квадраттарын шығару:

n = int(input("n санын енгіз: "))

for i in range(1, n + 1):
    print(i, "квадраты =", i**2)

3-мысал · Тек жұп сандарды шығару

n = int(input("n санын енгіз: "))

for i in range(2, n + 1, 2):
    print(i)

4-мысал · Мәтінді бірнеше рет шығару

for i in range(3):
    print("Python керемет!")
Шығыс:
Python керемет!
Python керемет!
Python керемет!

5-мысал · Тізімдегі элементтер санын санау

numbers = [3, 7, 2, 9, 4]

count = 0
for num in numbers:
    if num > 5:
        count = count + 1

print("5-тен үлкен элементтер саны:", count)

3-бөлім · Практика

Тапсырма 1 · Көбейту кестесінің қатары

Пайдаланушы енгізген сан үшін 1-ден 10-ға дейінгі көбейту кестесін for циклі арқылы шығар.

Кіріс Шығыс (қысқаша)
5 5 * 1 = 5 ... 5 * 10 = 50
7 7 * 1 = 7 ... 7 * 10 = 70

Тапсырма 2 · Жолдағы дауысты әріптер санын табу

Пайдаланушы енгізген жолдағы қазақ/латын дауысты әріптердің санын табатын программа жаз (a, e, i, o, u, ә, ө, ү, т.б.).

Кіріс (жол) Шығыс
"Salem" 2 (a, e)
"Python" 1 (o)

Тапсырма 3 · Тізім элементтерінің орташа мәні

Сандар тізімін беріп (немесе input арқылы алып), олардың орташа арифметикалық мәнін есептейтін программа жаз. 

Мысалы:
Кіріс: [4, 6, 10]
Шығыс: Орташа мәні: 6.67 (шекаралы түрде дөңгелектеуге болады)

4-бөлім · Mini-Quiz

    print("Hi")
