Бір өлшемді массив

Бұл сабақта біз бір өлшемді массив (Python тілінде – list тізімі) деген не екенін, оның түрлері мен ерекшеліктерін, элементтерді енгізу, қосу, іздеу тәсілдерін және практикалық есептерді қарастырамыз.

Сабақ соңында сен массивпен жұмыс істей алсаң, XP ұпайларың да автоматты түрде жиналады.

Оқу мақсаты

  • Массив ұғымын түсіну және өмірлік мысалдар келтіру;
  • Бір өлшемді массивтің (тізімнің) түрлері мен ерекшеліктерін білу;
  • Python тілінде массив (list) құру, элемент қосу, алу, енгізу тәсілдерін үйрену;
  • Массив элементтері бойынша іздеу, қосынды табу сияқты есептерді шығару;
  • Mini-quiz арқылы білімді тексеру.

1-бөлім · Теория

Массив деген не?

Массив – біртипті (бірдей типтегі) деректердің бірізділігі, олардың элементтері әртүрлі индекстермен ерекшеленіп, ортақ атаумен біріктіріледі.

Тұрмыста «тұрғын үй массиві», «орман массиві» сияқты ұғымдар бар. Бағдарламалауда массив ретінде сандар тізбегі қарастырылады: a₁, a₂, …, aₙ.

Массивтің түрлері

  • Бір өлшемді массив;
  • Екі өлшемді массив;
  • Көп өлшемді массив.

Массивтің типтері

  • Сандық;
  • Мәтіндік;
  • Символдық.

Массивтің ерекшеліктері

  • Элементтердің барлығы біртипті;
  • Элементтер бір атаумен аталады (мысалы, a, arr);
  • Элементтер жадта қатар орналасады.

Python-дағы бір өлшемді массив – тізім (list)

Python тілінде бір өлшемді массив көбіне list тізімі арқылы жүзеге асады. Тізім элементтері шаршы жақшада жазылады: 

numbers = [2, 4, 6, 8, 10]      # сандық массив
words   = ["alma", "banan"]     # мәтіндік массив
mixed   = [10, "python", 3.14]  # аралас типтер (Python рұқсат етеді)

Индекстеу (оң және теріс индекстер)

Python тізім элементтерін оң және теріс индекстер арқылы аламыз:

arr = [2, 3, 5, 7, 11]

# Оң индекстер
# Индекс:   0   1   2   3    4
# Элемент:  2   3   5   7    11

print(arr[0])   # 2
print(arr[2])   # 5
print(arr[4])   # 11

# Теріс индекстер
# Индекс:  -5  -4  -3  -2   -1
# Элемент:  2   3   5   7   11

print(arr[-1])  # 11 (соңғы элемент)
print(arr[-2])  # 7

Бос тізім құру

empty1 = []
empty2 = list()

print("Бос тізімдер:", empty1, empty2)
Консольдағы нәтиже:
Бос тізімдер: [] []

Тізімге элемент қосу (append)

append() әдісі тізім соңына жаңа элемент қосады:

arr = [1, 2, 3]
arr.append(4)
arr.append(5)

print("Тізім:", arr)
Консольдағы нәтиже:
Тізім: [1, 2, 3, 4, 5]

Срездер (кесінділер)

Тізімнің белгілі бір бөлігін алуға болады:

arr = [10, 20, 30, 40, 50]

print(arr[1:4])   # [20, 30, 40]
print(arr[:3])    # [10, 20, 30]
print(arr[2:])    # [30, 40, 50]

Массив элементтерін енгізу

Элементтерді бір жолға жазып, кейін тізімге айналдыру:

line = input("Сандарды бос орын арқылы енгіз: ")
parts = line.split()                 # ['1', '5', '10', '-3']
nums  = list(map(int, parts))       # [1, 5, 10, -3]

print("Тізім:", nums)
Мысалы, енгізу:
1 5 10 -3

Консольдағы нәтиже:
Тізім: [1, 5, 10, -3]

Қызықты ақпарат

  • Python-дағы тізімдер – динамикалық, яғни өлшемі программаның орындалу кезінде өзгереді.
  • Кейбір тілдерде (C, Java) массив өлшемі алдын ала беріледі, ал Python-да тізімге append() арқылы қалағаныңша элемент қоса аласың.
  • Тізімдер – олимпиадалық есептерде ең жиі қолданылатын құрылымдардың бірі.

2-бөлім · Мысалдар

Мысал 1 · Тізім элементтерінің қосындысы

Массивтегі барлық оң элементтердің қосындысын табайық:

nums = [3, -2, 10, 0, 7, -5]

total = 0
for x in nums:
    if x > 0:
        total += x

print("Оң элементтердің қосындысы:", total)
Консольдағы нәтиже:
Оң элементтердің қосындысы: 20

Мысал 2 · Дөңгелек сандар саны

Массивтегі дөңгелек сандар – нөлмен аяқталатын немесе 10-ға қалдықсыз бөлінетін сандар. Олардың санын табайық: 

nums = [10, 23, 40, 55, 100, 7, 30]

count_round = 0
for x in nums:
    if x % 10 == 0:
        count_round += 1

print("Дөңгелек сандар саны:", count_round)
Консольдағы нәтиже:
Дөңгелек сандар саны: 4

Мысал 3 · Массивті енгізу және орташа мән

Пайдаланушы енгізген сандардың орташа арифметикалық мәнін табамыз:

line = input("Сандарды енгіз (бос орын арқылы): ")
nums = list(map(int, line.split()))

s = sum(nums)
avg = s / len(nums)

print("Элементтер саны:", len(nums))
print("Қосынды:", s)
print("Орташа мән:", avg)
