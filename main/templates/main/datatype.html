Мәліметтер типтері

Бұл сабақта Python тіліндегі негізгі мәліметтер типтерін, олармен қандай амалдар орындауға болатынын және type() функциясын қолдануды үйренеміз.

Сабақты толық орындасаң, шамамен 100 XP жинайсың.

Оқу мақсаты

  • Мәліметтер типі деген ұғымды түсіну;
  • Python-дағы негізгі кіріктірілген мәліметтер типтерін ажырата білу;
  • type() функциясын қолданып, айнымалы типін анықтай алу;
  • Сандық, жолдық, тізім, кортеж, сөздік және логикалық типтермен қарапайым амалдар орындау;
  • Mini-Quiz арқылы білімін тексеру.

1-бөлім · Теория

Мәліметтер типі деген не?

Мәліметтер типі – айнымалы қандай мәндерді қабылдай алатынын және оларға қандай амалдар қолдануға болатынын сипаттайды. Яғни, тип – шаманың «табиғаты» туралы ақпарат.

Python тілінде көптеген мәліметтер типтері бар, бірақ мектеп деңгейінде біз негізінен мына кіріктірілген типтермен жұмыс жасаймыз:

  • int – бүтін сандар;
  • float – нақты сандар (жылжымалы нүктелі);
  • str – жолдық деректер (мәтін);
  • list – тізім;
  • tuple – кортеж;
  • dict – сөздік (кілт–мән жұптары);
  • bool – логикалық мәндер (True, False).

type() функциясы

Айнымалы немесе тұрақтының қай типке жататынын білу үшін type() функциясын қолданамыз. 

san = 5
print(type(san))   # <class 'int'>

name = "Aigerim"
print(type(name))  # <class 'str'>

Кез келген мәнді type() ішіне беруге болады: сан, жол, тізім, т.б.

Python-дағы сандық деректер типі

Python-да сандық деректердің негізгі екі типі бар:

  • int – бүтін сандар (-3, 0, 17);
  • float – нақты сандар, яғни үтірден кейінгі бөлігі бар сандар (-0.2, 3.1415, 2.0 т.б.).

Мысал:

san1 = 5        # int
san2 = 2.0      # float

print(san1, type(san1))
print(san2, type(san2))

# Арифметикалық амалдар
print(san1 + san2)
print(san1 * san2)

Python-дағы жолдық деректер типі (string)

Жол – бұл бір немесе екі тырнақша ішінде жазылған таңбалар жиынтығы. 

text1 = "Python"
text2 = 'Hello, world'

print(text1)
print(text2)

Жол элементтеріне индекс арқылы қол жеткіземіз (0-ден бастап), тілімдермен бөліп аламыз және операциялар қолдана аламыз: 

text = "Python"

print(text[0])     # 'P'   – бірінші символ
print(text[-1])    # 'n'   – соңғы символ
print(text[1:4])   # 'yth' – 1-ден 3-ке дейінгі символдар

print(text + " тілi")   # жолдарды қосу
print(text * 3)         # қайталау
print(text[::-1])       # кері жазу

Python-дағы тізім (list)

list – құрамдас мәліметтердің ең әмбебап түрі. Тізім элементтері тік жақша ішінде үтірмен бөлініп жазылады: [ ]. 

numbers = [1, 2, 3, 4]
mix = [1, "Python", 3.14, True]

print(numbers[0])      # 1
print(mix[1])          # "Python"

numbers[0] = 10        # тізім элементін өзгертуге болады
print(numbers)

Python-дағы кортеж (tuple)

tuple – тізімге ұқсайды, бірақ элементтері өзгермейді. Кәдімгі жақша қолданылады: ( ). 

coords = (10, 20)
print(coords[0])   # 10

# coords[0] = 30   # ҚАТЕ! Кортеж элементін өзгертуге болмайды

Егер тізімдегі мәндер өзгермеуі керек болса (мысалы, апта күндері), онда кортеж қолдану ыңғайлы және жадты тиімді пайдаланады.

Python-дағы сөздік (dict)

dict – кілт–мән (key–value) жұптарынан тұратын құрылым. Кілт арқылы мәнді оқуға болады. 

student = {"name": "Aigerim", "age": 15, "grade": 9}

print(student["name"])     # "Aigerim"

student["grade"] = 10      # мәнді өзгерту
print(student)

Сөздіктер { } фигуралық жақшасының ішінде кілт пен мәнді : арқылы бөліп жазады.

Python-дағы логикалық тип (boolean)

Логикалық тип екі ғана мән қабылдайды: True немесе False.

Негізгі салыстыру операторлары:

==, !=, <, >, <=, >=

Логикалық операторлар:

and, or, not 

a = 5
b = 3

print(a > b)              # True
print(a == b)             # False
print(a > 0 and b > 0)    # True
print(not (a > b))        # False

Тапсырма 1 · Типті анықтау

Төмендегі айнымалылардың типін дәптерге жаз және Python-да тексер:

a = 17
b = 3.5
c = "Python"
d = [1, 2, 3]
e = (4, 5, 6)
f = {"language": "Python", "level": "beginner"}
g = True

Әр айнымалыға type() қолдан:

print(type(a))
print(type(b))
# ... т.с.с.

Тапсырма 2 · Өз мысалың

  • Сандық типтерге (int, float) кем дегенде 2 мысал ойлап тап;
  • Жолдық типке 2 түрлі жол жаз;
  • Тізімге өзіңнің 3 сүйікті пәніңді енгіз;
  • Кортежге сыныптағы парта санын, қатар санын жаз;
  • Сөздікке өзің туралы шағын ақпарат енгіз (name, age, grade);
  • 2–3 логикалық өрнек құрып, нәтижесін шығар.
