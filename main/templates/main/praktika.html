IDE – мен танысу

Бұл сабақта Python бағдарламаларын жазуға арналған орта (IDE) не екенін, Python тілінің негізгі мүмкіндіктерін, алғашқы кодты және IDE-де жұмыс істеу қадамдарын үйренеміз.

Бұл сабақ соңында сен 100 XP жинап, деңгейді толық аяқтайсың.

Оқу мақсаты

  • Python тілінің ерекшеліктерін және қолдану салаларын түсіну;
  • IDE ұғымын түсіну және мысалдар келтіру;
  • Python-ды орнатып, алғашқы бағдарламаны жазу;
  • Кодты орындау/сақтау жолдарын меңгеру;
  • Mini-quiz арқылы білімді тексеру.

1-бөлім · Теория

Python тіліне кіріспе

Python — оқуға жеңіл, қуатты және әмбебап бағдарламалау тілі. Синтаксисі адамның табиғи тіліне ұқсайды, сондықтан бастаушыларға өте ыңғайлы.

Python деген не?

Python — жоғары деңгейлі, интерпретацияланатын тіл. Web, Data Science, жасанды интеллект, робототехника, автоматтандыру сияқты көптеген салаларда қолданылады.

Неге дәл Python?

  • Синтаксисі жеңіл, оқуға түсінікті;
  • Қате хабарламалары нақты және түсінікті;
  • Көп түрлі салаға жол ашады;
  • Бастаушыларға арналған ең қолайлы тілдердің бірі.

Python қайда қолданылады?

  • Web Development — Django, Flask;
  • Data Science — NumPy, Pandas;
  • Machine Learning — TensorFlow, PyTorch;
  • Automation — скрипттер, боттар, автоматтандыру;
  • Білім беру — мектепте, колледжде, университетте оқыту.

Алғашқы Python коды (Hello, Python!)

Тарихта көптеген программистер бірінші рет мына кодты жазған:

print("Hello, Python!")

Бұл код экранға мәтін шығарады. Python синтаксисінің қарапайымдылығын көрсетеді: бір ғана функция шақыру — және нәтиже дайын.

Python қалай жұмыс істейді?

Python — интерпретацияланатын тіл. Яғни код жол-жолымен оқылып, бірден орындалады. Біз файлда жазылған кодты іске қосқанда, Python интерпретаторы оны оқып, дереу орындайды.

Python үйренудің пайдасы

  • Ерте жастан IT саласында жұмыс табуға мүмкіндік;
  • Freelance жобалар арқылы қосымша табыс табу;
  • Data Science және AI бағыттарына жол ашады;
  • Олимпиада, жобалар, стартаптарда үлкен артықшылық береді.

IDE деген не?

IDE (Integrated Development Environment) — код жазуға, іске қосуға, қателерді табуға және жобаны басқаруға арналған кешенді орта. IDE ішінде редактор, терминал, дебаггер және басқа да құралдар біріктірілген.

Python-ды қайда жазамыз?

  1. PyCharm — кәсіби IDE, үлкен жобаларға ыңғайлы.
  2. VS Code — жеңіл редактор, кеңейтулер арқылы күшті IDE-ге айналады.
  3. Thonny — бастауыштарға арналған қарапайым IDE.
  4. Онлайн IDE — браузерден бірден жаза беруге болады (OnlineGDB, Replit т.б.).

IDE-дегі алғашқы код

print("Salem, Python!")

IDE-ні ашып, осы кодты теріп, іске қосып көр. Экранда «Salem, Python!» мәтіні шығады.

Пайдаланушыдан атын сұрайтын программа

Келесі бағдарлама қолданушыдан атын сұрап, сәлемдеседі:

name = input("Атың кім? ")
print("Сәлем,", name)

2-бөлім · Практика және орнату

Python орнату (үйдегі компьютерге)

Python-ды ресми сайттан жүктеуге болады: python.org → Downloads бөлімін таңда.

  • Жүйеңе сәйкес нұсқаны таңда (Windows, macOS, Linux);
  • Орнату кезінде “Add Python to PATH” деген жеріне құсбелгі қой;
  • Орнатуды аяқтаған соң, IDE-ден Python-ды ашып, код жазып көр.

Тапсырма 1 · Алғашқы программа

Python файлына төмендегі кодты жазып, іске қос:

print("Мен Python тілін оқуды бастадым")

Тапсырма 2 · Екі қатарлы код

IDE-де төмендегі кодты жазып, нәтижесін көр:

print("Мен Python үйреніп жүрмін!")
print(2 + 3)

Тапсырма 3 · Өзің туралы бағдарлама

Өзің туралы 3 жол ақпарат шығаратын бағдарлама жаз:

print("Менің атым ...")
print("Мен ... сыныптамын")
print("Мен Python үйренемін")

Кодты іске қосқанда қате шықса, жолдағы символдарды (тырнақша, жақша, бос орын) мұқият тексер.

3-бөлім · Mini-Quiz

Төмендегі сұраққа жауап бер (ауызша, дәптерде немесе мұғалімнің платформасында):

Сұрақ:

print("Hello")

Осы код орындалғанда экранда не пайда болады?

  1. Hello
  2. "Hello"
  3. print("Hello")

Дұрыс жауапты таңда да, неліктен солай екенін түсіндір.

4-бөлім · Сабақты аяқтау және үй тапсырмасы

Сабақты аяқтаудың чек-листі

  • Python тілінің не екенін және қайда қолданылатынын түсінемін;
  • Компьютерге Python орната аламын;
  • IDE ашып, қарапайым бағдарламаны жаза аламын;
  • print() және input() функцияларының не істейтінін білемін;
  • Mini-quiz сұрағына жауап бердім.

Үй тапсырмасы

  • Үйдегі компьютерге Python және бір IDE (PyCharm, VS Code немесе Thonny) орнат;
  • «Мен және Python» тақырыбында кем дегенде 5 жолдық бағдарлама жаз (әр жолда өзің туралы немесе мақсаттарың туралы ақпарат болсын);
  • Файлды сақтап, келесі сабақта көрсет;
  • Қосымша: ата-анаңа немесе досыңа Python туралы қысқаша түсіндіріп бер.
