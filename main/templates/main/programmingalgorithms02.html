Кірістірілген шарттар (nested if)

Бұл сабақта шарт ішінде тағы бір шарт жазуды, яғни кірістірілген if конструкциясын қарастырамыз. Осындай құрылым күрделірек логиканы жазуға көмектеседі.

Сабақты толық орындасаң, осы тақырып бойынша логикалық ойлау дағдың жақсы қалыптасады.

Оқу мақсаты

  • if блогының құрылымын қайталау;
  • Кірістірілген шарт ұғымын түсіну;
  • Кірістірілген шартпен программа құру (баға қою, температура, үшбұрыш түрі т.б.);
  • Mini-quiz арқылы білімді тексеру.

1-бөлім · Теория

Жай шартты операторды еске түсіру

Қарапайым if–elif–else конструкциясы былай жазылады:

x = int(input("Сан енгіз: "))

if x > 0:
    print("Оң сан")
elif x < 0:
    print("Теріс сан")
else:
    print("Нөл")

Мұнда әрбір if, elif, else блогының ішінде тек бір деңгейдегі командалар орындалады.

Кірістірілген шарт деген не?

Кірістірілген шарт (nested if) — бір if блогының ішіне басқа if немесе if–elif–else жазу. Яғни «шарттың ішінде тағы бір шарт» болады.

Жалпы түрі:

if шарт1:
    # 1-деңгей
    if шарт2:
        # 2-деңгей
        ...
    else:
        ...
else:
    ...

Мұндай құрылым мынадай ситуацияларда пайдалы:

  • алдымен үлкен категорияны анықтау (мысалы, «емтихан тапсырды ма?»);
  • содан кейін ішкі категорияға бөлу (баға қою, қосымша хабарлама т.б.).

· Емтихан нәтижесі

Оқушының алған ұпайына байланысты алдымен «өту/өтпеуін», содан кейін әртүрлі баға деңгейін шығарайық: 

score = int(input("Ұпайыңды енгіз (0–100): "))

if score >= 50:
    print("Емтиханнан өттің!")

    if score >= 90:
        print("Баға: 5 (өте жақсы)")
    elif score >= 75:
        print("Баға: 4 (жақсы)")
    else:
        print("Баға: 3 (қанағаттанарлық)")
else:
    print("Өкінішті, емтиханнан өтпедің.")

Мұнда сыртқы if score >= 50 шартынан кейін ғана ішкі if–elif–else жұмыс істейді.

· Температура және киім

Сыртқы шарт – «жаңбыр бар ма?», ішкі шарт – «температура қандай?»:

rain = input("Жаңбыр жауып тұр ма? (иә/жоқ): ")
t = int(input("Температура: "))

if rain == "иә":
    print("Қолшатыр ал!")

    if t < 5:
        print("Қалың күртеше ки.")
    else:
        print("Жеңіл күртеше жеткілікті.")
else:
    print("Жаңбыр жоқ.")

    if t >= 25:
        print("Өте жылы, жеңіл киін.")
    else:
        print("Қалыпты ауа райы.")

2-бөлім · Мысалдар

1-мысал · Үшбұрыш түрін анықтау

Алдымен үшбұрыш құрыла ма, содан кейін оның түрін (теңқабырғалы, теңбүйірлі, әрқабырғалы) анықтаймыз. 

a = int(input("a қабырғасы: "))
b = int(input("b қабырғасы: "))
c = int(input("c қабырғасы: "))

# Алдымен үшбұрыш құрыла ма?
if a + b > c and a + c > b and b + c > a:
    print("Үшбұрыш құрылады.")

    # Ішкі шарттар: үшбұрыш түрі
    if a == b == c:
        print("Теңқабырғалы үшбұрыш.")
    elif a == b or a == c or b == c:
        print("Теңбүйірлі үшбұрыш.")
    else:
        print("Әрқабырғалы үшбұрыш.")
else:
    print("Үшбұрыш құрылмайды.")

2-мысал · Логин және роль

Пайдаланушының логині дұрыс болса, содан кейін оның ролін тексереміз. 

username = input("Логин: ")
role = input("Рөлің (admin/user): ")

if username == "student01":
    print("Логин дұрыс.")

    if role == "admin":
        print("Қош келдің, админ!")
    else:
        print("Қарапайым қолданушы ретінде кірдің.")
else:
    print("Логин қате.")

3-бөлім · Практика

Тапсырма 1 · Баға және пікір

Пайдаланушыдан 0–100 аралығында ұпай енгізуді сұра. Егер ұпай >= 50 болса – «емтиханнан өттің» деп жаз, содан кейін кірістірілген шартпен қосымша пікір шығар:

  • 90–100 – «өте жақсы жұмыс»;
  • 75–89 – «жақсы нәтиже»;
  • 50–74 – «өту балын алдың, бірақ тағы қайталау керек».
Кіріс (ұпай) Шарт Шығыс (хабарлама)
95 score >= 90 "Емтиханнан өттің! Өте жақсы жұмыс!"
80 75 <= score < 90 "Емтиханнан өттің! Жақсы нәтиже."
60 50 <= score < 75 "Емтиханнан өттің! Тағы қайталау керек."
40 score < 50 "Өтпедің."

Тапсырма 2 · Жыл мезгілі және ауа райы

Пайдаланушыдан жыл мезгілі ("қыс", "көктем", "жаз", "күз") және температура мәнін сұрайтын программа жаз.

Сыртқы шарт – мезгіл, ішкі шарт – температура диапазоны.

Кіріс Шарт Шығыс
мезгіл = "қыс", t = -10 мезгіл == "қыс", t < 0 "Аязды қыс."
мезгіл = "жаз", t = 30 мезгіл == "жаз", t >= 25 "Ыстық жаз."

Тапсырма 3 · Логин, пароль және доступ

Пайдаланушыдан логин және пароль сұрайтын программа жаз. Егер екеуі де дұрыс болса, ішкі шартпен оның ролін (admin/user) тексер.

Кіріс Шарттар Шығыс
login = "user", pass = "1234", role = "user" login=="user" және pass=="1234" "Кіру сәтті. Қарапайым қолданушы."
login = "admin", pass = "admin", role = "admin" login=="admin" және pass=="admin" "Кіру сәтті. Админ панеліне доступ."

4-бөлім · Mini-Quiz

Төмендегі 6 сұраққа жауап бер. Содан кейін «Тексеру» батырмасын бас.

1) Кірістірілген шарт деген не?

2) Төмендегі кодта ішкі шарт қай жерде орналасқан?

if x >= 0:
    print("оң немесе нөл")

    if x % 2 == 0:
        print("жұп")
    else:
        print("тақ")

3) Кірістірілген шартты қолданудың басты себебі?

4) Қай код үлгісі кірістірілген шартты дұрыс көрсетеді?

5) Кірістірілген шарттарда ең маңыздысы не?

6) Кірістірілген шарт қай кезде орындалады?

Сабақты қорытындылау

  • Кірістірілген шарттың не екенін түсіндім;
  • Сыртқы және ішкі if блоктарын шатастырмай жаза аламын;
  • Ұпай, баға, ауа райы, логин сияқты өмірлік мысалдармен жұмыс істедім;
  • Mini-quiz сұрақтарына жауап бердім.
