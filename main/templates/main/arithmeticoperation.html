Арифметикалық амалдардың жазылу ережелері

Бұл сабақта Python тіліндегі негізгі арифметикалық операцияларды, олардың орындалу тәртібін, салыстыру операторларын және math модулін қолдануды үйренеміз.

Сабақты толық орындасаң, шамамен 100 XP жинайсың.

Оқу мақсаты

  • Арифметикалық операциялардың не екенін түсіну;
  • Python тіліндегі негізгі арифметикалық операторларды ажырата білу;
  • Орындалу тәртібін (приоритетін) дұрыс қолдану;
  • Салыстыру операторлары мен логикалық нәтиже (True/False) түсіну;

1-бөлім · Арифметикалық операциялар

Арифметикалық операциялар деген не?

Арифметикалық операциялар – сандармен орындалатын есептеу операциялары. Python тілінде негізгі арифметикалық операциялар: қосу, азайту, көбейту, бөлу, қалдықты есептеу, бүтін бөлікті есептеу және дәрежелеу. Python тіліндегі арифметикалық операциялардың орындалу тәртібі математикадағы орындалу тәртібіне ұқсас.

Python-дағы негізгі арифметикалық операторлар

  • + – қосу;
  • - – азайту;
  • * – көбейту;
  • / – бөлудің нәтижесі (нақты сан float);
  • // – бүтін бөлікті бөлу (төмен қарай дөңгелектейді);
  • % – қалдықты есептеу (mod);
  • ** – дәрежелеу (power).

Мысалдар

a = 7 + 3       # 10
b = 7 - 3       # 4
c = 7 * 3       # 21
d = 7 / 3       # 2.3333333...
e = 7 // 3      # 2  (бүтін бөлігі)
f = 7 % 3       # 1  (қалдық)
g = 2 ** 3      # 8  (2-нің 3-дәрежесі)

print(a, b, c, d, e, f, g)

Орындалу тәртібі (оператор приоритеті)

Операциялар орындалу реті математикаға ұқсас: алдымен дөңгелек жақша, сосын дәрежелеу, содан кейін көбейту/бөлу, соңында қосу/азайту. 

n1 = 2 + 3 * 4        # 2 + 12 = 14
n2 = (2 + 3) * 4      # 5 * 4 = 20

print(n1)
print(n2)

Жақшаны дұрыс қою – есептің дұрыс шығуына өте маңызды.

2-бөлім · Салыстыру операторлары

Салыстыру операторлары екі мәнді немесе айнымалыны салыстырады және логикалық нәтиже ретінде True немесе False қайтарады.

Негізгі салыстыру операторлары

  • == – тең бе?
  • != – тең емес пе?
  • > – үлкен бе?
  • < – кіші ме?
  • >= – үлкен немесе тең бе?
  • <= – кіші немесе тең бе?

Мысалдар

a = 5
b = 3

print(a > b)    # True
print(a == b)   # False
print(a != b)   # True
print(a >= 5)   # True
print(a < 0)    # False

Салыстырудың нәтижесі логикалық типке (bool) жатады және шартты операторларда (if) жиі қолданылады.

3-бөлім · math модулі

Математикалық көптеген қосымша функциялар math модуліне біріктірілген. Python-дағы модуль – ішінде дайын функциялары бар файл.

Модульді қолдану үшін import командасымен қосамыз: 

import math

math модулінің негізгі функциялары

  • math.pow(num, power) – num санын power дәрежесіне шығарады;
  • math.sqrt(num) – num санының квадрат түбірін табады;
  • math.ceil(num) – нақты санды ең жақын үлкен бүтін санға дейін дөңгелектейді;
  • math.floor(num) – нақты санды ең жақын кіші бүтін санға дейін дөңгелектейді.

Мысалдар

import math

print(math.pow(2, 3))     # 8.0
print(math.sqrt(16))      # 4.0
print(math.ceil(3.14))    # 4
print(math.floor(3.99))   # 3

Бұл функциялар күрделірек есептер мен формулаларды программалау кезінде өте ыңғайлы.

4-бөлім · Практика

Тапсырма 1 · Арифметикалық өрнектер

Төмендегі өрнектердің мәнін алдымен дәптерде қолмен есепте, кейін Python-да тексер: 

a = 7 + 4 * 2
b = (7 + 4) * 2
c = 15 // 4
d = 15 % 4
e = 2 ** 5
f = (3 + 5) * 2 - 10 / 5

print(a, b, c, d, e, f)

Тапсырма 2 · Салыстыру

Әр өрнектің нәтижесі True немесе False болатынын болжа, содан соң тексер:

print(7 > 3)
print(10 == 2 * 5)
print(5 != 5)
print(8 <= 4 * 2)
print(3 + 2 > 3 * 2)

Тапсырма 3 · math модулімен жұмыс

Төмендегі программаны IDE-де немесе онлайн компиляторда орында:

import math

r = 5
circle_length = 2 * math.pi * r      # шеңбер ұзындығы
circle_area = math.pi * r ** 2       # шеңбер ауданы

print("Ұзындығы:", circle_length)
print("Ауданы:", circle_area)

Өзгеше радиус мәндерін қойып көр және нәтиженің қалай өзгеретінін бақыла.
